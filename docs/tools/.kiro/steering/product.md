# Product Context

`nextjs-starter-template` is a production-ready starter for teams building modern web apps with Next.js.

It emphasizes:

- fast project bootstrapping
- strict TypeScript and stable architecture
- built-in quality gates (lint/type/tests/build)
- security and dependency hygiene
- GitHub automation for PR checks and releases

Primary user value:

- reduce setup time
- reduce manual release overhead
- provide a consistent contributor workflow
